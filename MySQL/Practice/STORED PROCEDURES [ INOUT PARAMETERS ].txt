# STORED PROCEDURES [ INOUT PARAMETERS]


Q1 Create a procedure named increase_salary that accepts an INOUT salary parameter and increases it by 1000.

Ans:

mysql> delimiter #

mysql> create procedure increase_salary ( inout plussalary decimal(10, 2) )
    -> begin
    -> set plussalary = plussalary + 1000 ;
    -> end #

mysql> delimiter ;

Q2 Execute the procedure for a salary of 30000 and display the updated salary.

Ans:

mysql> set @plussalary = 10000;

mysql> call increase_salary(@plussalary);

mysql> select @plussalary;
