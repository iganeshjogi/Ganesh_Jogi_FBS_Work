Q1) Create a procedure print_1_to_10 using a WHILE loop.

    -> declare i int default 1;

    -> set i = i + 1;

    -> end #

#### LABEL LOOP


Q1) Create a procedure using a label and print numbers 1 to 5.

Ans:

mysql> delimiter #

mysql> create procedure lable_loop_demo()
    -> begin
    -> declare i int default 1;
    -> print_1_5 : loop
    -> select i;
    -> set i = i + 1;
    -> if i > 5 then
    -> leave print_1_5;
    -> end if;
    -> end loop print_1_5;
    -> end #

mysql> delimiter ;

#### REPEAT LOOP


Q1) Create a procedure repeat_1_to_5 using a REPEAT loop.

Ans:

mysql> delimiter #

mysql> create procedure repeat_1_to_5()
    -> begin
    -> declare i int default 1;
    -> repeat
    -> select i;
    -> set i = i + 1;
    -> until i = 6
    -> end repeat;
    -> end #
