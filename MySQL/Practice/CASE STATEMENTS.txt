# CASE STATEMENTS

Q1) Write an SQL query to display Employee Name, Salary, and Salary_Status using the following conditions:

Salary greater than or equal to 50000 → High Salary
Otherwise → Low Salary

Ans:

mysql> selectemp_ name, salary,
    -> case
    -> when salary >= 50000 then 'High Salary'
    -> else 'Low Salary'
    -> end as Salary_status
    -> from employee;

Q2) Write an SQL query to display Student Name, Marks, and Result using the following conditions:

Marks greater than or equal to 75 → Distinction
Marks between 60 and 74 → First Class
Marks between 35 and 59 → Pass
Marks less than 35 → Fail

Ans:

mysql> select Name, Marks,
    -> case
    -> when Marks >= 75 then 'Distinction'
    -> when Marks between 60 and 74 then 'First Class'
    -> When Marks between 35 and 59 then 'Pass'
    -> When Marks < 35 then 'Fail'
    -> end as Result
    -> from students;

Q6) Write an SQL query to update the Salary of employees using a CASE statement with the following conditions:

Employees from Pune → Increase salary by 10%
Employees from Mumbai → Increase salary by 15%
All other employees → Increase salary by 5%

Ans:

mysql> update employees
    -> set salary = salary +
    -> case
    -> when city = 'Pune' then salary*0.1
    -> when city = 'Mumbai' then salary*0.15
    -> else salary*0.05
    -> end ;
