# TRIGGER

Q1) Create Trigger names salary_backup_trigger to save old salary.

Ans:

mysql> delimiter #

mysql> create trigger salary_backup_trigger
    -> before update
    -> on employee
    -> for each row
    -> begin
    -> insert into salary_backup
    -> values
    -> (old.id, old.salary, curdate() );
    -> end#

mysql> delimiter ;


Q2) Create a trigger named book_backup. 
	BEFORE UPDATE
	On books
Save:
	OLD.book_no
	OLD.price
	CURDATE()
Into a table named book_backup.

Ans:

mysql> delimiter #

mysql> create trigger book_backup
    -> before update
    -> on books
    -> for each row
    -> begin
    -> if old.price <> new.price then
    -> insert into book_backup values (old.book_no,old.price,curdate());
    -> end if ;
    -> end#

mysql> delimiter ;
