Q1) Write an SQL query to create a view named Book_List that displays the following columns
from Books table:
Book_No, Book_Name, Author_Name, Cost, Category.

Ans:

mysql> create view Book_List as
    -> select book_no, book_name, author, price, category
    -> from books;

Q2) Write an SQL query to display all records from the view Book_List.

Ans:

mysql> select * from book_list;

Q3) Write an SQL query to create a view named Issued_Books that displays details of all books
that are currently issued
(books whose Return_Date is NULL).

Ans:

mysql> create view Issued_Books as
    -> select *
    -> from issue_details
    -> where return_date is null;

Q4) Write an SQL query to display all records from the view Issued_Books.

Ans:

mysql> select * from Issued_Books;

Q5) Write an SQL query to create a view named Member_Penalty that displays Member_Id,
Member_Name, Penalty_Amount
for members whose Penalty_Amount is greater than 0.

Ans:

mysql> create view Member_Penalty as
    -> select member_id, member_name, penalty_amount
    -> from members
    -> where penalty_amount > 0;

Q6) Write an SQL query to display all records from the view Member_Penalty.

Ans:

mysql> select * from member_penalty;

Q7) Write an SQL query to update the Penalty_Amount of a member using the Member_Penalty view.

Ans:

mysql> update member_penalty
    -> set penalty_amount = 15
    -> where member_id = 108;

Q8) Write an SQL query to create a view named Book_Issue_Details that displays: 
Book_Name, Member_Name, Issue_Date, Return_Date
by joining Books, Member, and Issue tables.

Ans:

mysql> create view Book_issue_details as
    -> select b.book_name, m.member_name, i.issue_date, i.return_date
    -> from books as b
    -> inner join
    -> issue_details as i
    -> on b.book_no = i.book_no
    -> inner join
    -> members as m
    -> on i.member_id = m.member_id;

Q9) Write an SQL query to display Book_Name, Member_Name, Issue_Date from the view Book_Issue_Details.

Ans:

mysql> select book_name, member_name, issue_date
    -> from Book_issue_details;
